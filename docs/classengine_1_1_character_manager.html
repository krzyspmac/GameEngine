Detailed Description

Manages a list of character representations. Allows loading of the characters from specific json files. Keeps a list of character representations available for future use.

CharacterRepresentation * CharacterManager::LoadCharacter ( std::string  jsonFilename)

Loads or gets the character previously loaded. jsonFilename must exist in the game files as well as any other files referenced by the json. Loaded characters are kept in memory by the manager. No need to take ownership of the intances.
