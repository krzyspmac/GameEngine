Detailed Description

The LightManager keeps a list of light objects as well as data cache for various different shaders.

Create a light. The result is being managed by the manager.

Parameters
type- type of falloff; either "linear" or "exponential"
color- light color, rgb
ambientIntensity- background intensity when not inside param diffuseSize
position- x, y in game coordinates
diffuseSize- size of the light area in game coordinates
diffuseIntensity- the amount of light shine onto the scene wihtin the diffuseSize
-- Create a light with a "linear" falloff, white, with a 0.1 ambient value
-- positioned at 400, 350 with a diffuse size of 250 and diffuse intensity
-- of 1.
light = LightManager:CreateLight("linear", 1, 1, 1, 0.1, 400, 350, 250, 1)
