Detailed Description

Provides a way to animate a value from point a to point b along a certain curve. ValueAnimator does not animate object properties (like "alpha"). It only provides a way to animate values.

See also
ValueAnimatorFactory to create a concrete property animator and
PropertyAnimatorFactory to create a property animator.

The ValueAnimator takes min, max, the curve function, the duration and two callbacks: the update function and the finish function.

The script is resposible for memory management.
