Detailed Description

Main accessor for files. Returns FileMemoryBufferI items so different part of the engine can load the data from a byte stream.

Loads up a buffer stream for a specific filename. If the FileAccessI was setup up with a directory a FileMemoryBufferStreamFromFile will be returned. In case of a packaed resource file this will return FileMemoryBufferStreamFromBundle. Memory ownership is passed to the caller.
